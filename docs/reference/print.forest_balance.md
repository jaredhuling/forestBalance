# Print a forest_balance object

Displays a concise summary of the forest balance fit, including the ATE
estimate, sample sizes, effective sample sizes, and a brief covariate
balance overview.

## Usage

``` r
# S3 method for class 'forest_balance'
print(x, ...)
```

## Arguments

- x:

  A `forest_balance` object returned by
  [`forest_balance`](https://jaredhuling.github.io/forestBalance/reference/forest_balance.md).

- ...:

  Ignored.
